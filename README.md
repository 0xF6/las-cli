##### Flags

in env: 
```yaml
- CLI_EMOJI_USE    : 0 - disable cli use emoji
- CLI_COLOR_USE    : 0 - disable cli use coloring
```
example:
```powershell
$env:CLI_EMOJI_USE=0;./lascli --help
```
